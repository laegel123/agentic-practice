# 에이전트 효율 점수표

같은 [벤치마크 과제](benchmark-tasks.md)를 **전(레거시) / 후(입힌 뒤)** 동일하게 시키고 과제별로 기록한다.
검증하려는 가설:

> 에이전틱+하네스 적용 후 → **성공률 ↑, 턴·토큰·시간 ↓, 권한 프롬프트 ↓, 자가검증 ↑, 회귀 ↓**

## 측정 항목 (과제마다)

| 항목 | 의미 | 출처 |
|------|------|------|
| 성공 | 수용 기준/정답지 충족 (O/X/부분) | answer-key 대조 |
| 턴 수 | 사용자↔에이전트 왕복 수 | 세션 카운트 |
| 도구 호출 수 | 툴 콜 개수 | 세션 |
| 토큰 | 입력+출력 토큰 | Claude Code `/cost` 또는 하단 표시 |
| 소요시간 | 체감 wall-clock | 스톱워치 |
| 권한 프롬프트 | 권한 승인 요청 횟수 | 관찰 |
| 추가 질문 | 에이전트가 사람에게 되물은 횟수 | 관찰 |
| 자가검증 | 빌드/테스트로 스스로 확인했는가 (O/X) | 관찰 |
| 회귀 | 다른 기능/테스트를 깼는가 (O/X 또는 건수) | 재빌드·재테스트 |
